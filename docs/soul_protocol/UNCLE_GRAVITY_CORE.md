# UNCLE GRAVITY CORE
## Strategic Development Principles

 These principles govern the architectural and aesthetic decisions of all projects developed under the **UncleGravity / Antigravity** name for Codex Archai Systems.

---

## 1. FREE-FIRST ARCHITECTURE (FFA)
- **Zero-Op Infrastructure**: Prioritize platforms with high-value free tiers (GitHub Actions for compute, Cloudflare Pages for hosting).
- **Resource Minimalism**: Design systems to run within Free Tier rate limits (e.g., 10s synthesis delays, JSON-based storage instead of heavy databases).
- **Scale-Ready**: Infrastructure must be capable of upgrading to paid tiers without structural rewrites.

## 2. CYBER-NOIR AESTHETIC
- **Tactical Visuals**: All interfaces must feel like Strategic Terminals or Intelligence Command Centers.
- **Precision**: Glassmorphism (blur/translucency), high-contrast mono fonts, and deep-space palettes (`#05070a`).
- **Function over Decoration**: Every visual element must serve a data purpose.

## 3. SOUL PROTOCOL (ANTI-AMNESIA)
- **Persistent Context**: Every project must contain its own "Neural Link" (docs/soul_protocol/).
- **Linear Chronicles**: Maintain a stage-by-stage record of design pivots and technical decisions.
- **Agent Integration**: Documentation must be "Agent-Optimized" (concise, structured, technical) to allow for seamless context handover between AI instances.

## 4. AUTHORITATIVE MINIMALISM
- **Communication Tone**: Precise, technical, and authoritative. Avoid marketing fluff or unnecessary personification.
- **Single-Source Intelligence**: Synthesize complex data into clear, fact-based insights.
