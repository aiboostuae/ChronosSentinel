# UNCLE GRAVITY CORE
## Strategic Operating System (v2.2.0)

This document codifies the non-negotiable principles governing all architecture and development within Codex Archai Systems (CAS).

---

### 0. THE PRIME DIRECTIVE: SECURITY > MIMO
**When principles conflict, Security ALWAYS wins.**
- **Protection**: Client data, IP (trade secrets, schemas), and infrastructure secrets are the highest priority.
- **Zero-Trust**: No "convenience" bypasses. Exposing internal cdn/backend URLs in frontend code is a violation.
- **When in doubt**: Choose security over simplicity.

---

### 1. FIRST PRINCIPLES (FP)
- **Root Cause Analysis**: Solve the fundamental problem, not the symptom.
- **Deconstruction**: Tracing assumptions back to baseline physics/logic.
- **Clarity over Complexity**: Understand the "WHY" before implementing the "HOW."

---

### 2. MIMO (MINIMUM INPUT → MAXIMUM OUTPUT)
- **Leverage**: Maximize scale with the least relative effort.
- **Eliminate Waste**: No redundancy. Single source of truth for all data and logic.
- **FFA (Free-First Architecture)**: Use high-leverage free tiers (GitHub Actions, Cloudflare) for production infrastructure to minimize overhead.
- **Scaling Rule**: If a solution doesn't scale across the entire system, refactor or reject.

---

### 3. MC-LHI (MINIMUM COST TO LESSEN HUMAN INTERVENTION)
- **Strategic Automation**: Spend time/effort/money upfront to automate human overhead out of the system forever.
- **The 48-Hour SLA**: Automation must enable a 48-hour period from Payment to Live Site.
- **Maintenance-Free**: Build logic that "runs itself" (e.g., self-correcting intake, automated synthesis).

---

### 4. CYBER-NOIR AESTHETIC (SENTINEL STYLE)
- **Interface Authority**: Designs must feel like Architecture Labs or Strategic Terminals (e.g., Palantir, Anduril).
- **Glassmorphism**: Use translucent glass-panels with backdrop blur and luminous cyan accents.
- **Precision**: High-contrast typography and tactical dark-mode palettes (`#05070a`).

---

### 5. SOUL PROTOCOL (ANTI-AMNESIA)
- **Neural Link**: Maintain the `docs/soul_protocol/` directory as a project's "Soul."
- **Context-Locking**: Versioned chronicles and technical manifests must allow any AI Agent to resume work with zero context loss.
