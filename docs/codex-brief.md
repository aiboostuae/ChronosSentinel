# Codex Brief: Chronos Sentinel (CS)

## Overview
Chronos Sentinel is an AI-assisted open-source intelligence dashboard that transforms global news streams into source-compared, confidence-scored, historically archived event briefings.

## Architecture
- **Layer A**: RSS Ingestion
- **Layer B**: Article Extraction
- **Layer C & D**: Clustering & Synthesis via Gemini API
- **Layer E**: Snapshot Archiving
- **Frontend**: Single-page static frontend in `/public` using HSL-tailored colors, dynamic navigation, and localized region filtering.
